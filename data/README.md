# Data to treat

## treat one single file:<br>
<code>python rmr-calculation.py --folder data/<data-folder-name></code><br>
<code>python rmr-calculation.py --folder data/<data-folder-name> --verbose</code> (with verbose)<br>
<code>python rmr-calculation.py</code> (without argument, enter: "<data-folder-name>")

## recursive execution over all files included in data/:
<code>python rmr-calculation.py --folder data --recursive</code><br>
<code>python rmr-calculation.py --folder data --verbose --recursive</code><br>
